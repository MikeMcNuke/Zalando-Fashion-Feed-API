Enter your username and password

If you don't have a username at zalando, please register one at this place: https://www.zalando.de/login!

API Call

type: "GET",
url: "https://api.dz.zalan.do/auth/token",
headers: {
     Accept: "application/x.zalando.myfeed+json;version=2",
     Authorization: "Basic" + base64encodedCredentials
},

Retrieve User Profile

With the authentification token you are now able to retrieve the whole profile of the logged in user. Press 'Retrieve Profile' to get the profile.

For testing purpose we offer you an prepared profile. It's the ME profile and replaces the profile ID in all API calls, which are personalized. Click on 'Use 'ME Profile' to use this example profile instead of your own.

API Call

type: "GET",
url: "https://api.dz.zalan.do/auth/me",
headers: {
     Accept: "application/x.zalando.myfeed+json;version=2",
     Authorization: userProfile.access_type + " " + userProfile.access_token
},

General Topbrands

Get the general top brands from Zalando.

API Call

type: "GET",
url: "https://api.dz.zalan.do/feeds/MALE/sources/brand_reco/streams/top/items",
headers: {
     Accept: "application/x.zalando.myfeed+json;version=2",
     Authorization: userProfile.access_type + " " + userProfile.access_token
},

Personal Brand Recommendations

Get personal brand recommendations for the user, based on his profile.You first have to get the user profile.

API Call

type: "GET",
url: "https://api.dz.zalan.do/feeds/" + userProfile.feed_id + "/sources/brand_reco/streams/personalized/items",
headers: {
     Accept: "application/x.zalando.myfeed+json;version=2",
     Authorization: userProfile.access_type + " " + userProfile.access_token
},
